## Testing Website

A tiny, framework-free starter site. Everything runs with just HTML, CSS, and vanilla JS.

### Files

- `index.html` — page structure
- `styles.css` — colors, layout, responsive tweaks
- `script.js` — theme toggle, demo toast, fake contact submit

### Run locally

From the project folder:

```sh
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Customize quickly

- Update copy and links in `index.html`.
- Swap the accent color by changing `--primary` in `styles.css`.
- Remove the demo toast or form handlers in `script.js` if not needed.
